# 🚀 Goal Tracker API

[](https://fastapi.tiangolo.com/)
[](https://www.python.org/)

A lightweight, high-performance REST API built with **FastAPI** to manage your personal goals. This project allows you to create, track, update, and even export your goals to a CSV file for offline viewing.

## 👤 Author

  * **GitHub:** [@NithishProgrammer](https://www.google.com/search?q=https://github.com/NithishProgrammer)
  * **Repo Name:** `goal-tracker-api`

-----

## 🛠️ Features

  * **Full CRUD:** Create, Read, Update, and Delete goals.
  * **Unique Identification:** Automatic generation of IDs for every goal.
  * **Granular Updates:** Specific endpoints to update names, deadlines, or descriptions individually.
  * **CSV Export:** Download your entire goal list as an `Active_goals.csv` file.
  * **Auto-Documentation:** Interactive Swagger UI provided by FastAPI.

-----

## 📥 Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/NithishProgrammer/goal-tracker-api.git
    cd goal-tracker-api
    ```

2.  **Install dependencies:**

    ```bash
    pip install fastapi uvicorn
    ```

3.  **Run the server:**

    ```bash
    uvicorn main:app --reload
    ```

    The API will be available at `http://127.0.0.1:8000`.

-----

## 📖 API Documentation

Once the server is running, visit `/docs` to see the interactive Swagger UI.

### 1\. Goal Object Structure

Every goal sent to the API must follow this JSON format:

```json
{
  "name": "Learn FastAPI",
  "description": "Build an open-source goal tracker",
  "start_d": "2024-01-01",
  "end_d": "2024-02-01",
  "deadline": "2024-03-01"
}
```

### 2\. Endpoints Summary

| Method | Endpoint | Description |
| :--- | :--- | :--- |
| **GET** | `/goals` | Retrieve all active goals. |
| **POST** | `/post` | Create a new goal (Auto-assigns `Goal_id`). |
| **DELETE** | `/del/{Goal_id}` | Remove a goal by its unique ID. |
| **PUT** | `/update/{Goal_id}/name` | Update only the name of a goal. |
| **PUT** | `/update/{Goal_id}/deadline` | Update only the deadline. |
| **PUT** | `/update/{Goal_id}/description` | Update only the description. |
| **PUT** | `/update/{Goal_id}/start_d` | Update the start date. |
| **PUT** | `/update/{Goal_id}/end_d` | Update the end date. |
| **GET** | `/export` | Download all goals as `Active_goals.csv`. |

-----

**Goal ID can't be changed (autogenerated)**

## 🚀 Usage Examples

### Create a Goal (POST)

**URL:** `/post`  
**Body:**

```json
{
  "name": "Morning Gym",
  "description": "Workout for 1 hour",
  "start_d": "06:00 AM",
  "end_d": "07:00 AM",
  "deadline": "Everyday"
}
```

### Update a Goal Name (PUT)

**URL:** `/update/12345678/name?name=Evening%20Gym`  
*This will search for ID `12345678` and update the title.*

### Exporting Data

Simply navigate to `http://127.0.0.1:8000/export` in your browser. The API will generate a CSV file on the server and trigger an automatic download.
